Differentiation (Intro)

Standard Derivatives

  • d/dx(xⁿ) = n·xⁿ⁻¹
  • d/dx(sin x) = cos x
  • d/dx(cos x) = −sin x
  • d/dx(tan x) = sec²x
  • d/dx(cot x) = −cosec²x
  • d/dx(sec x) = sec x·tan x
  • d/dx(cosec x) = −cosec x·cot x
  • d/dx(eˣ) = eˣ
  • d/dx(aˣ) = aˣ ln a
  • d/dx(ln x) = 1/x
  • d/dx(logₐ x) = 1/(x ln a)
  • d/dx(√x) = 1/(2√x)

Inverse Trigonometric Derivatives

  • d/dx(sin⁻¹x) = 1/√(1−x²)
  • d/dx(cos⁻¹x) = −1/√(1−x²)
  • d/dx(tan⁻¹x) = 1/(1+x²)
  • d/dx(cot⁻¹x) = −1/(1+x²)
  • d/dx(sec⁻¹x) = 1/(x√(x²−1))
  • d/dx(cosec⁻¹x) = −1/(x√(x²−1))

Rules of Differentiation

  • d/dx(u ± v) = du/dx ± dv/dx
  • d/dx(uv) = v·du/dx + u·dv/dx
  • d/dx(u/v) = (v·du/dx − u·dv/dx)/v²
Chain Rule:
d/dx[f(g(x))] = f′(g(x))·g′(x)
Used when one function is inside another.
Variable Power Rule:
d/dx(mⁿ) = mⁿ · d/dx(n ln m)
Used when both base and exponent depend on x.

Tangent and Normal

The derivative dy/dx represents the slope of the tangent at a point.

  • Tangent: y − y₁ = (dy/dx)(x − x₁)
  • Normal: y − y₁ = −1/(dy/dx)(x − x₁)
  • Parallel to x-axis → dy/dx = 0
  • Equal angles with axes → dy/dx = ±1
